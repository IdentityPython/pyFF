pyFF

A SAML Federation Appliance

Get the code on Github!
 

In short...

PyFF is a simple, yet complete SAML metadata aggregator with...

  • Fully customizable processing pipelines.
  • Ability to retrieve, analyze, transform, sign and publish SAML metadata.
  • Operate in batch or online mode using embedded HTTP server.
  • Provide a full MDX implementation.
  • Make use of PKCS#11 tokens and HSMs for key protection.
  • SAML 2.0 Discovery Service using twitter bootstrap adaptive CSS
