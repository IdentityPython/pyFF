pyFF


A SAML Metadata Appliance - Get Started

Get the code on Github

 
 
 

In short...

PyFF - short for python Federation Feeder - is a simple, yet complete SAML metadata aggregator. Feature highlights include:

  • Fully customizable processing pipelines in yaml.
  • Easy to retrieve, analyze, transform, sign and publish SAML metadata.
  • Operate in batch or online mode using embedded HTTP server.
  • Provide a full MDX implementation.
  • Make use of PKCS#11 tokens and HSMs for key protection.
  • SAML 2.0 Discovery Service using bootstrap 3.1 adaptive CSS.
  • Discojuice JSON metadata compatibility.
