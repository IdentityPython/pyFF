pyFF

A SAML Federation Appliance

Get the code on Github!

In short...

PyFF is a simple, yet complete SAML metadata aggregator with support for:

  • Customizable processig pipelines.
  • Fetch, analyze, transform, sign and publish SAML metadata.
  • Batch or online mode using embedded HTTP server.
  • MDX implementation.
  • Use PKCS#11 tokens and HSMs for key protection.
